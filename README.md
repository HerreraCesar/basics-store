## ¿Como usar la aplicación de forma local? <a name="4"></a>

Este proyecto requiere **<a href="https://nodejs.org/" target="_blank">Node.js</a>** para su ejecución.

```
git clone https://github.com/HerreraCesar/basics-store.git
```
```
cd basics-store
```
```
npm install
```
```
npm start
```
